- #### craete file
` cat > file_name.txt `
- #### view content of file
` cat file_name.txt `
- #### view content of file in reverse
` tac file_name.txt `
- #### view content of file with line number
` cat -n file_name.txt `
- #### view content of more files
` cat *.txt `
- ### copy file1 content into file2 (file2 data will replaced)
` cat file1.txt > file2.txt `
- ### copy file1 content into file2 (file1 will append in file2)
` cat file1.txt >> file2.txt `
